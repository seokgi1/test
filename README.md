# PhoneShop (간단한 핸드폰 판매 데모)

로컬에서 간단한 핸드폰 판매 페이지입니다. 브라우저로 열어 제품 목록을 보고 장바구니 기능을 테스트할 수 있습니다.

사용법:

1. 이 폴더에서 `index.html` 파일을 더블클릭해 브라우저로 엽니다.
2. 검색창/필터로 제품을 찾고, `장바구니에 추가` 버튼으로 담아보세요.
3. `장바구니` 버튼을 눌러 담긴 항목과 총 합계를 확인할 수 있습니다.

- 서버 연동(상품/주문 API)
- 결제 연동
- 반응형 이미지 및 실제 상품 데이터

Google AdSense 설정 (간단 안내):

- `index.html`의 `<head>`에 있는 `data-ad-client="ca-pub-XXXXXXXXXXXX"` 값을 본인의 AdSense `ca-pub-...` ID로 교체하세요.
- 페이지 내 예제 광고 블록의 `data-ad-slot`을 본인의 광고 단위 ID로 바꾸세요.
- 로컬(파일://)에서 광고가 바로 보이지 않을 수 있습니다. 배포된 도메인(예: GitHub Pages)에 도메인을 연결하고 AdSense 계정에서 도메인을 등록/승인해야 실제 광고가 나옵니다.
- 자세한 설정은 Google AdSense 문서를 참조하세요.

- 서버 연동(상품/주문 API)
- 결제 연동
- 반응형 이미지 및 실제 상품 데이터
